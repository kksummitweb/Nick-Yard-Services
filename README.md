## ğŸ¨ Customization

### Adding Images

1. Place your images in the `images/` directory
2. Replace placeholder divs in HTML files with actual images:

```html
<!-- Replace placeholder -->
<div class="image-placeholder">
    <i class="fas fa-image"></i>
    <p>Company Image</p>
</div>

<!-- With actual image -->
<img src="images/your-image.jpg" alt="Description">
```

### Updating Content

- **Contact Information**: Update phone, email, and address in all HTML files
- **Services**: Edit service descriptions in `services.html`
- **About Content**: Modify company story and team information in `about.html`
- **Gallery**: Add project images in `gallery.html`

### Color Scheme

Colors can be customized in `css/style.css`:

```css
:root {
    --primary-color: #2c3e50;    /* Main brand color */
    --secondary-color: #3498db;  /* Accent color */
    --accent-color: #e74c3c;     /* Call-to-action color */
}
```
